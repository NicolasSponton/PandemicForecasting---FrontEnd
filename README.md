## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone NicolasSponton/PandemicForecasting---FrontEnd
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
2. run App
   ```sh
   npm run dev
   ```
